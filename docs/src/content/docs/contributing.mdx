# Contributing

See the repository guide in `CONTRIBUTING.md` for full details. Highlights:

- Setup
  - Node.js 18+, `npm install`, then `npm run dev`.
- Conventions
  - Tailwind for styles; Radix UI for primitives.
  - Lint with `npm run lint`.
  - Conventional Commits; small focused PRs.
- Docs
  - This site is built with Astro + Starlight under `/docs`.
